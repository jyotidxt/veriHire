# 🛡️ VeriHire

> AI-powered Chrome Extension & SaaS that helps job seekers evaluate the trustworthiness of LinkedIn job postings before applying.

## 📌 Overview

VeriHire is a Full Stack AI application consisting of a Chrome Extension and a modern SaaS dashboard.

Instead of labeling jobs as **"Fake"**, VeriHire analyzes multiple trust signals and provides an explainable **Trust Score**, helping users make informed decisions.

---

## ✨ Features

### Chrome Extension

* Detect LinkedIn job pages
* Floating **Verify Job** button
* One-click job analysis
* Modern popup interface
* Fast analysis

### AI Analysis

* Trust Score (0–100)
* Risk Level
* Company verification signals
* Suspicious wording detection
* URL safety analysis
* AI-generated recommendations

### Dashboard

* Scan history
* Saved jobs
* Analytics
* User profile
* Settings
* Report suspicious jobs
